# 🔊 UT Transducer Selector Toolkit

A free, offline reference tool for ultrasonic thickness gauging — built for quality inspectors and NDT technicians.

No installation. No login. No internet required. Just open the HTML file in any browser.

---

## What's included

Two standalone HTML files — one in English, one in Spanish — each containing four tools:

### 1. Evident Transducer Selector
Select material, measurement technique (direct contact, immersion, or corrosion), and thickness range to get a ranked list of compatible Microscan transducers with coverage-based recommendations.

- 26 materials: metals, plastics, ceramics, composites, rubber, and more
- Contact and immersion techniques with mode-coded results (Mode 1 / 2 / 3)
- Corrosion survey mode with 43 dual-element transducers across 8 application types
- Temperature ratings, probe tip diameter, and frequency for corrosion transducers

### 2. Sound Velocity Calculator
Reference sound velocities for 27 materials from Evident's official data, plus a live thickness calculator using the fundamental equation:

```
T = V × (t / 2)
```

Includes a temperature effect guide — critical for plastics and rubber where velocity changes significantly with temperature.

### 3. Quick Reference
- Measurement modes 1, 2, 3 and Dual Element — with real equipment waveform screens
- Transducer type guide: contact, delay line, immersion, dual element
- Frequency selection table

### 4. NDT Glossary
25 key terms for ultrasonic thickness gauging, searchable in real time. Available in English and Spanish.

## How to use

1. Download the HTML file for your preferred language
2. Open it in any modern browser (Chrome, Firefox, Safari, Edge)
3. No server, no dependencies, no connection needed

To share with your team: upload to [Netlify Drop](https://app.netlify.com/drop) and share the URL — the tool runs entirely in the browser.

## Coverage

**Metals** — Aluminum (sheet, plate, tubing), Brass/Bronze, Copper, Cast iron, Carbon steel (sheet, plate, pipe), Cast steel, Stainless steel, Titanium

**Plastics** — Acrylic, Polyamide (Nylon), LDPE, HDPE/PVC, Polystyrene, Teflon (PTFE)

**Other materials** — Ceramic, Fiber epoxy composite, Fiberglass, Rubber, Silicon, Soft tissue, Water and liquids

**Corrosion applications** — Steel pipe/tank/vessel, Boiler tube/pressure vessel, Cast iron, Small diameter pipe, Severely thinned/pitted wall, High temperature in-service, Internal oxide scale, Single element/EMAT

---

## Notes

- All transducer data and measurement ranges are indicative. Actual performance depends on part geometry, surface condition, and instrument setup.
- Material is assumed to be at ambient temperature unless otherwise stated. Sound velocity and attenuation change with temperature — always recalibrate at actual test temperature, especially for plastics and rubber.
- This tool is not a substitute for appropriate operator training.
